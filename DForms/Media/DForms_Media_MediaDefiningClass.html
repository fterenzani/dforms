Class DForms_Media_MediaDefiningClass

Description
Description | Descendents | Methods (details)

A base class that provides media statically, or optionally per instance.

Subclasses define their media on a class basis by overriding the ``media`` static method. Instances may simply supply their own ``media`` member variable, which will short circuit the dynamically generated value (since ``__get()`` won't be called at all for ``media``).

Media is inherited by subclasses even if they do not define their own media.

		static media (line 138)
	
 
	
	

Defines the media for the class.


Subclasses should override this method to return an associative array  of media paths. Keys should be the type of media (currently only ``js``  and ``css``) and values should be relative or absolute URLs. Media is  inherited by subclasses *automatically*, and subclasses shoud *not*  call the parent method.
.. note:: This method must remain public until PHP5.3 since it must be     accessed from ``call_user_func()``. It should eventually be protected.
