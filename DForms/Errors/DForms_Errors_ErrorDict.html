Class DForms_Errors_ErrorDict

Description
Description | Methods (details) Constants

A dictionary of errors with special output rendering methods.

The errors contained in the dictionary may be accessed using familiar associative array syntax:

  1.  $messages = array('test' => 'A test error occurred.');
  2.  
  3.  // Create an error dictionary from existing array.
  4.  $errors = new DForms_Errors_ErrorDict($messages);
  5.  
  6.  // Loop over each error in the dictionary.
  7.  foreach ($errors as $name => $msg) {
  8.      echo sprintf('Found error %s: %s', $name, $msg);
  9.  }
  10.  
  11.  // Remove an error from the dictionary.
  12.  unset($errors['test']);
  13.  
  14.  // Check for errors.
  15.  if (count($errors)) {
  16.      do_something($errors);
  17.  }
  18.  
  19.  // Output errors rendered as html.
  20.  echo $errors;
